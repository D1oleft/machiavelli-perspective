# Niccolò Machiavelli 著作与系统性长文调研

> 调研日期：2026-06-17
> 说明：一手文献指马基雅维利本人所写；二手文献指后世学者的总结、评注或传记。
> 可信度标注：高 = 学术界广泛接受的基本事实；中 = 主流观点但有少量争议；低 = 有显著学术分歧。

---

## 一、出版书籍与核心著作

### 1.1 《君主论》（Il Principe / The Prince）

| 项目 | 内容 |
|------|------|
| 写作时间 | 1513年（主体），1513-1514年间完成 |
| 首次出版 | 1532年（作者去世后5年），罗马教廷许可出版 |
| 献词对象 | 洛伦佐·德·美第奇（Lorenzo de' Medici） |
| 体裁 | 政治论著（dedicated treatise），共26章 |
| 文献类型 | **一手文献** |
| 可信度 | 高 |

**核心论点：**
- 君主应以"实效真理"（verità effettuale）而非想象中的理想国为行动依据（第15章）
- 与其被爱不如被畏惧，但切不可被憎恨（第17章）
- 君主必须学会"不行善"——当必要性（necessità）要求时（第15章、第18章）
- 命运（fortuna）主宰人事一半，另一半由virtù决定（第25章）
- 武装的先知胜过 unarmed 的先知——必须以自己的武力为基础（第6章）
- 新君主比世袭君主面临更大挑战，需通过个人能力而非制度获得合法性（多个章节反复出现）
- 应效法切萨雷·博尔贾（Cesare Borgia）的果断行动（第7章）
- 建立殖民地优于军事占领，因为成本低且怨恨小（第3章）
- 君主必须避免被蔑视和憎恨（第16-19章连续论证）
- 必要时必须使用残暴（crudeltà bene usate），且应一次性完成而非渐进式（第8章）

**结构概要：**
- 第1-11章：君主国的类型（世袭、混合、新、教会）
- 第12-14章：军事问题
- 第15-23章：君主的品质与行为准则
- 第24-25章：意大利君主为何失国、命运问题
- 第26章：呼吁解放意大利

**矛盾与争议点：**
- 第18章"论守信"中提出君主应当守信但更应学会不守信，此章与第15章的"实效真理"存在张力——究竟是彻底的amorality还是有底线的pragmatism，学界有分歧
- 全书整体论调是为专制君主提供手册，但第26章充满爱国主义激情，两者之间存在明显张力
- 是否为讽刺作品（ironic reading）：部分学者（如J.G.A. Pocock、Garrett Mattingly）认为此书可能是讽刺或劝谏性质，而非字面意义上的专制指南；但主流观点倾向于按字面理解

**二手文献参考：**
- Harvey Mansfield, *Machiavelli's Virtue* (1996)——强调virtù概念的复杂性
- Quentin Skinner, *Machiavelli: A Very Short Introduction* (2000)——将马基雅维利置于共和主义传统中理解
- Isaiah Berlin, "The Originality of Machiavelli" (1972)——提出马基雅维利开创了价值多元主义
- Leo Strauss, *Thoughts on Machiavelli* (1958)——强调马基雅维利的隐微写作（esoteric writing）

### 1.2 《论李维罗马史前十书》（Discorsi sopra la prima deca di Tito Livio / Discourses on Livy）

| 项目 | 内容 |
|------|------|
| 写作时间 | 约1517年（1513-1519年间断续写作，与《君主论》写作有重叠） |
| 首次出版 | 1531年，罗马 |
| 献词对象 | 扎诺比·布翁德尔蒙蒂（Zanobi Buondelmonti）和科西莫·鲁切莱（Cosimo Rucellai） |
| 体裁 | 政治论著，对李维《罗马史》前十卷的评注，分3卷共142章 |
| 文献类型 | **一手文献** |
| 可信度 | 高 |

**核心论点：**
- 共和政体优于君主政体（第1卷第2章）——与《君主论》形成明显张力
- 人民比君主更稳定、更少犯错、更能适应变化（第1卷第58章）
- 宗教是社会秩序和政治控制的关键工具（第1卷第11-15章）
- 冲突（disordine）——罗马贵族与平民的斗争——是罗马自由的来源（第1卷第4-6章）
- 所有政治体都有自然衰退周期（第1卷第2章、第5卷第1章）
- 军事是共和国的根基，公民军队优于雇佣军（第2卷多处）
- 毒杀、欺骗等手段在政治中可以是必要的（第3卷第2-7章讨论Mucius Scaevola等人）
- 先例（esempi）的力量大于理论论证
- 混合政体（君主、贵族、民主的混合）是最稳定的制度安排（第1卷第2章、第6章）
- 个人自由（vivere libero）是共和国的核心目标

**结构概要：**
- 第1卷（46章）：论罗马城邦的建立、制度、宗教、冲突
- 第2卷（33章）：论罗马的军事制度与对外扩张
- 第3卷（49章）：论个人德行与制度衰落
- 附录：论阴谋（未完成）

**矛盾与争议点：**
- **核心矛盾**：《君主论》似乎是君主制的指南，而《论李维》是共和主义的宣言。学界对此有多种解释：
  - "两面论"：马基雅维利对君主制和共和制持工具性态度，视情境而定
  - "互补论"：《君主论》讨论特殊情况（建国），《论李维》讨论常规治理
  - "共和优先论"：《论李维》代表他的真正信念，《君主论》是权宜之计或求职书
- 第1卷第26章和第3卷中对罗慕路斯杀弟行为的辩护，与共和主义理想存在紧张关系

**二手文献参考：**
- J.G.A. Pocock, *The Machiavellian Moment* (1975)——将马基雅维利放在共和主义传统中
- Gisela Bock, Quentin Skinner, Maurizio Viroli eds., *Machiavelli and Republicanism* (1990)
- Harvey Mansfield, *Machiavelli's New Modes and Orders* (1979)——对《论李维》的逐章解读

### 1.3 《战争的艺术》（Dell'arte della guerra / The Art of War）

| 项目 | 内容 |
|------|------|
| 写作时间 | 约1519-1520年 |
| 首次出版 | 1521年，佛罗伦萨（Giunti出版社）——**马基雅维利生前唯一出版的著作** |
| 献词对象 | 洛伦佐·斯特罗齐（Lorenzo Strozzi） |
| 体裁 | 对话体军事论著，分7卷 |
| 文献类型 | **一手文献** |
| 可信度 | 高 |

**核心论点：**
- 公民军队（cittadino esercito）优于雇佣军和外国援军
- 应效法古罗马的军事制度，恢复步兵的核心地位
- 军事与政治不可分割——军事制度的腐败必然导致政治腐败
- 将领必须同时具备军事才能和政治判断力
- 骑兵应服从步兵，而非相反
- 训练和纪律比个人勇武更重要
- 军事工程和地形知识是将领的必备技能
- 战争有其"艺术"（arte），可以像其他技艺一样被系统学习和教授

**对话设定：**
- 背景：鲁切莱花园（Orti Oricellari），1516年后的实际聚会地点
- 主要对话者：法布里齐奥·科隆纳（Fabrizio Colonna，真实人物，雇佣兵将领）与佛罗伦萨贵族青年
- 形式模仿西塞罗的对话体，但论点基本由科隆纳一人主导

**矛盾与争议点：**
- 此书讨论的是军事问题，但论证方式带有强烈的政治理论色彩，与《君主论》和《论李维》中的军事论点高度一致
- 科隆纳本人是雇佣兵将领，但书中却系统性地批判雇佣军制度——这一设定本身就带有讽刺意味（或马基雅维利有意借用其权威性）

### 1.4 《佛罗伦萨史》（Istorie fiorentine / Florentine Histories）

| 项目 | 内容 |
|------|------|
| 写作时间 | 1520-1525年 |
| 首次出版 | 1532年，罗马 |
| 委托方 | 教皇克莱门特七世（Giulio de' Medici）——美第奇家族 |
| 体裁 | 历史著作，分8卷，涵盖1215-1492年 |
| 文献类型 | **一手文献** |
| 可信度 | 高（作为历史叙述有偏见，但作为马基雅维利思想的一手资料可信度高） |

**核心论点：**
- 佛罗伦萨的内部冲突源于贵族与平民（grandi与popolo）的结构性对立
- 美第奇家族的崛起代表了一种"伪装的专制"——表面共和、实质独裁
- 外部征服（如米兰的维斯孔蒂家族）往往被用作内部政治斗争的工具
- 制度设计（ordini）比个人德行更能决定政治稳定
- 历史应以因果关系而非编年体方式书写

**矛盾与争议点：**
- 受美第奇家族委托撰写，因此对美第奇的批评较为含蓄，需要"读出字里行间"的意思
- 与《论李维》中的共和主义立场一致，但在此书中更为隐晦
- 第8卷（至1492年洛伦佐·德·美第奇去世）未涉及当代政治，刻意回避

### 1.5 喜剧作品

#### 《曼陀罗》（La Mandragola / The Mandrake）

| 项目 | 内容 |
|------|------|
| 写作时间 | 约1518年 |
| 首次出版 | 1524年 |
| 体裁 | 五幕喜剧 |
| 文献类型 | **一手文献** |
| 可信度 | 高 |

**核心主题：**
- 讽刺佛罗伦萨社会的道德堕落——教会、学者、贵族全部被嘲笑
- 主角卡利马科（Callimaco）通过欺骗和操纵成功获得他人之妻——与《君主论》中的权术主题平行
- 修道士提莫窦（Fra Timoteo）被收买配合骗局——直接讽刺教会腐败
- 尼西亚博士（Messer Nicia）是愚蠢的典型代表——饱学之士被利用
- 讽刺结论：恶行获得了好结果，甚至获得了社会认可

**与政治理论的关联：**
- 与《君主论》共享"必要性驱使道德妥协"的主题
- 展示马基雅维利对"实效真理"的文学化表达
- 被认为是意大利文艺复兴喜剧的巅峰之作

#### 《克莉齐娅》（Clizia）

| 项目 | 内容 |
|------|------|
| 写作时间 | 约1525年 |
| 首次演出 | 1525年1月13日，佛罗伦萨鲁切莱花园 |
| 体裁 | 五幕喜剧（改编自普劳图斯的《卡西娜》） |
| 文献类型 | **一手文献** |

**核心主题：**
- 老年人的情欲与年轻人的欲望之间的冲突
- 通过奴隶克莉齐娅的视角展示权力与服从关系
- 结局强调秩序的恢复——但通过欺诈手段实现

#### 《贝丽扎》（La Clizia / Belisarga，另有其他短剧）

- 马基雅维利还写有其他短剧和狂欢节歌曲（canti carnascialeschi），包括《驴子颂》（Decennale primo, Decennale secondo）等韵文作品

### 1.6 《驴的喜剧》（La commedia delle comedie / comedies相关）

马基雅维利的文学作品还包括：
- **《自驴之口》（Belfagor arcidiavolo）**：中篇小说（novella），约1518年，讲述魔鬼下凡体验婚姻生活，讽刺佛罗伦萨的社会风气

## 二、长篇政治论述、外交书信与备忘录

### 2.1 主要政治论述

#### 《论我们的语言》（Discorso o dialogo intorno alla nostra lingua）

| 项目 | 内容 |
|------|------|
| 写作时间 | 约1514-1516年（有争议） |
| 体裁 | 语言学论著 |
| 文献类型 | **一手文献** |
| 可信度 | 中（作者归属有争议，部分学者认为是马基雅维利所写） |

**核心论点：**
- 但丁的托斯卡纳方言应被视为意大利标准语言的基础
- 语言是政治统一的工具

#### 《论小洛伦佐死后佛罗伦萨的政务》（Discursus florentinarum rerum post mortem iunioris Laurentii Medices）

| 项目 | 内容 |
|------|------|
| 写作时间 | 1520年 |
| 背景 | 小洛伦佐·德·美第奇去世后，马基雅维利为教皇克莱门特七世撰写的备忘录 |
| 体裁 | 政治备忘录 |
| 文献类型 | **一手文献** |

**核心论点：**
- 佛罗伦萨应恢复共和制，但需在美第奇家族的影响下进行
- 建议建立一个新的政治制度，融合共和元素与美第奇的权威
- 详细讨论了如何通过制度改革防止寡头政治

#### 《论改革佛罗伦萨国家的方案》（Riforma dello stato di Firenze）

| 项目 | 内容 |
|------|------|
| 写作时间 | 1520年 |
| 体裁 | 制度设计提案 |
| 文献类型 | **一手文献** |

**核心内容：**
- 为佛罗伦萨设计新的政治制度
- 建议扩大公民参与，但保留美第奇的领导地位
- 体现了马基雅维利在理论与实践之间的折中

#### 《论如何处理比萨叛乱者的备忘录》（Del modo di trattare i popoli della Valdichiana ribellati）

| 项目 | 内容 |
|------|------|
| 写作时间 | 1503年 |
| 体裁 | 政策备忘录 |
| 文献类型 | **一手文献** |

**核心论点：**
- 应对比萨采取强硬但聪明的策略
- 讨论了如何通过经济手段和政治手段的结合来控制叛乱地区
- 早期已显示出马基雅维利对"必要性"政治的思考

### 2.2 外交书信与报告

马基雅维利在1498-1512年间担任佛罗伦萨共和国第二国务厅长官（Cancelliere della Seconda Cancelleria）和"十人委员会"（Dieci di Balia）秘书，留下了大量外交文书。

#### 主要外交报告（Legazioni, Commissarie）

| 出使任务 | 时间 | 主要内容 | 文献类型 |
|----------|------|----------|----------|
| 出使法国国王路易十二世 | 1500年 | 报告法国外交政策和意大利局势 | **一手文献** |
| 出使切萨雷·博尔贾 | 1502年10月-1503年1月 | 亲历博尔贾在伊莫拉的行动，成为《君主论》第7章的主要素材 | **一手文献，极高价值** |
| 出使罗马教廷（朱利乌斯二世） | 1503年、1506年 | 报告教皇选举和教廷政策 | **一手文献** |
| 出使神圣罗马帝国皇帝马克西米利安一世 | 1507-1508年 | 报告帝国的财政和军事弱点 | **一手文献** |
| 出使曼托瓦的弗朗切斯科·贡扎加 | 多次 | 军事观察 | **一手文献** |

**重要特点：**
- 外交报告中已经出现"virtù"、"fortuna"等核心概念的雏形
- 对切萨雷·博尔贾的观察报告（1502-1503年）是《君主论》第7章的直接素材
- 报告风格客观、冷静，带有分析性，与理论著作的风格一致

**二手文献参考：**
- 《马基雅维利全集》（Edizione nazionale delle opere di Niccolò Machiavelli）——权威学术版本
- Francesco Guicciardini对马基雅维利书信的批评——提供了同时代人的评价

马基雅维利的私人书信是他思想的重要来源，特别是1513年被流放后的书信。

#### 关键书信

| 弗朗切斯科·韦托里 | 1513-1527年（多封） | 讨论政治、文学、个人生活 | **一手文献** |

| 乔万尼·韦罗利（Giovanni Vernacci） | 多封 | 个人事务和农业生活 | **一手文献** |
| 路易吉·阿拉曼尼（Luigi Alamanni） | 多封 | 讨论文学和政治 | **一手文献** |

**重要书信内容：**
- 1513年12月10日致韦托里的信：马基雅维利描述自己"穿上宫廷礼服"与古人对话，是理解《君主论》写作动机的关键文献
- 书信中多次讨论命运的无常和个人在逆境中的应对——与理论著作的主题一致
- 流放后的书信带有明显的挫败感和渴望被召回的愿望

## 三、反复出现的核心论点（真信念）

以下论点在马基雅维利的多部著作中出现3次以上，可视为其"真信念"：

### 3.1 人性本恶/人性可操纵

**出现频率：** 《君主论》第17、18、25章；《论李维》第1卷第3、29章，第3卷多处；《曼陀罗》全剧；外交报告多处

**表述：**
- "关于人类，一般可以这样说：他们是忘恩负义的、容易变心的、伪装的、怯懦的、贪财的"（《君主论》第17章）
- 人民的欲望（appetito）比君主的欲望更危险，但可以通过制度加以引导

**可信度：** 高（一手文献中反复明确表述）

### 3.2 雇佣军不可靠，公民军队是共和国的根基

**出现频率：** 《君主论》第12-13章；《论李维》第2卷多处；《战争的艺术》全书；外交报告中多次提及

**表述：**
- 雇佣军"既不团结、又无纪律、不讲信义"（《君主论》第12章）
- 应效法古罗马，建立由本国公民组成的军队

**可信度：** 高

### 3.3 宗教是政治工具

**出现频率：** 《君主论》第6、11、18章；《论李维》第1卷第11-15章；《佛罗伦萨史》多处

**表述：**
- 罗马的宗教制度（ordini religiosi）对维持国家秩序至关重要
- 教皇的世俗权力损害了意大利的政治统一
- 摩西、居鲁士等建国者都利用了宗教

**可信度：** 高

### 3.4 命运（Fortuna）与个人能动性（Virtù）的辩证关系

**出现频率：** 《君主论》第7、25章；《论李维》第2卷第1、29章，第3卷多处；《战争的艺术》；书信中多次提及

**表述：**
- "命运是我们行动的半个主宰，但她留下另一半或近乎一半由我们支配"（《君主论》第25章）
- 命运像河流，平时可以筑堤预防，但洪水来临时无法阻挡
- Virtù是对fortuna的回应和对抗

**可信度：** 高

### 3.5 必要性（Necessità）高于道德规范

**出现频率：** 《君主论》第15、17、18章；《论李维》第3卷第2-7章；《曼陀罗》

**表述：**
- "如果一个人在一切事情上都想以善良自持，那么他在许多不善良的人当中必然毁灭"（《君主论》第15章）
- 博尔贾的残暴行为是"必要的残暴"（necessitate）

**可信度：** 高

### 3.6 新秩序/新制度（Nuovi ordini）的建立需要强力手段

**出现频率：** 《君主论》第6章；《论李维》第1卷第9、18章，第3卷多处

**表述：**
- 建立新秩序是最困难的事业，因为旧秩序的受益者会成为敌人
- 必须依靠个人的virtù或军队来推行新制度
- 摩西、罗慕路斯、提修斯都使用了暴力手段

**可信度：** 高

### 3.7 混合政体优于单一政体

**出现频率：** 《论李维》第1卷第2、6章；《佛罗伦萨史》；《论改革佛罗伦萨国家的方案》

**表述：**
- 斯巴达、罗马、威尼斯都采用了混合政体
- 单一政体（君主、贵族、民主）都有自然退化倾向

**可信度：** 高

### 3.8 内部冲突可以是制度活力的来源

**出现频率：** 《论李维》第1卷第4-6章；《佛罗伦萨史》多处

**表述：**
- 罗马贵族与平民的冲突（disordine）是罗马自由和制度发展的来源
- 但佛罗伦萨的冲突过于激烈，破坏了制度——关键在于冲突是否在制度框架内

**可信度：** 高（但与通常的"秩序优于混乱"的政治理论形成对比，是马基雅维利的独到之处）

## 四、自创术语与核心概念

### 4.1 Virtù（德/能力/力量）

**定义：** 这是马基雅维利最复杂、最具原创性的概念。它不是传统意义上的"美德"（virtue），而是一种政治行动的能力，包括果断、适应性、远见、勇气、对情境的判断力。

**使用语境：**
- 《君主论》第25章：virtù是对抗fortuna的能力
- 《君主论》第6章：摩西、居鲁士、罗慕路斯、提修斯都有virtù
- 《论李维》第1卷第9章：一个有virtù的公民可以改变共和国的制度
- 《战争的艺术》：将领必须具备virtù

**学术争议：**
- 经典解释（Viroli, Skinner）：virtù是公民德行（civic virtue）
- 曼斯菲尔德解释（Mansfield）：virtù是与基督教美德对立的异教德行
- 伯林解释（Berlin）：virtù是价值多元主义下的独立价值

**可信度：** 高（概念的复杂性本身是学术共识）

### 4.2 Fortuna（命运/运气）

**定义：** 不可控的外部力量和机遇，包括历史时机、自然灾害、人事变迁。马基雅维利不否认其存在，但强调人可以通过virtù来应对。

**使用语境：**
- 《君主论》第25章：最系统的论述
- 《君主论》第7章：博尔贾的命运起伏
- 《论李维》第2卷第29章：命运在战争中的作用

**与古典传统的关系：**
- 继承了罗马命运女神（Fortuna）的形象
- 但比古代斯多葛派更强调人对命运的反抗能力

### 4.3 Necessità（必要性）

**定义：** 迫使行动者采取非常手段的客观情境。不是抽象的哲学概念，而是具体的政治压力。

**使用语境：**
- 《君主论》第15章：必要性迫使君主"学会不行善"
- 《君主论》第18章：必要时可以不守信
- 《论李维》第3卷：在紧急情况下，独裁权力是必要的

### 4.4 Occasio（时机/机遇）

**定义：** 历史提供的特定时刻，行动者必须迅速把握。与fortuna相关但不完全相同——fortuna是持续的力量，occasio是具体的瞬间。

**使用语境：**
- 《君主论》第25章：命运是女性，需要用暴力征服
- 《论李维》第3卷：时机的把握是政治成功的关键

### 4.5 Modo vivere（生活方式/存在方式）

**定义：** 特定政治共同体的根本特征和习惯。新君主必须理解并尊重（或改变）人民的modo vivere。

**使用语境：**
- 《君主论》第3、5章：征服新领地后是否改变其modo vivere，决定了统治的成败

### 4.6 Ordini与Ordinare（制度/制度安排）

**定义：** 政治制度的根本结构，包括法律、制度设计、权力分配。比个人品质更重要的是制度（ordini）的质量。

**使用语境：**
- 《论李维》第1卷：罗马的ordini是其成功的关键
- 《佛罗伦萨史》：佛罗伦萨的制度缺陷导致了内乱

### 4.7 Exemplo（范例/先例）

**定义：** 历史先例是政治行动的指南。马基雅维利系统性地使用历史案例来支持论点。

**方法论意义：**
- 这是马基雅维利政治学的方法论基础——通过历史案例的归纳来得出一般性结论
- 与亚里士多德的演绎法和柏拉图的理念论形成对比

## 五、对古典思想家的引用和改造

### 5.1 李维（Titus Livius）

**引用频率：** 最高——《论李维》的直接对话对象

**引用方式：**
- 对《罗马史》前十卷逐章评注
- 但不是忠实的文本解读，而是选择性引用和改造
- 将李维的历史叙述转化为政治理论

**改造要点：**
- 李维本人是罗马共和主义的歌颂者，马基雅维利继承了这一传统
- 但马基雅维利比李维更强调冲突（disordine）的积极意义——李维倾向于将冲突视为需要克服的问题，马基雅维利则视其为制度发展的动力
- 马基雅维利从李维那里借用了大量具体的制度案例，但赋予它们新的理论意义

### 5.2 色诺芬（Xenophon）

**引用频率：** 中等

**引用方式：**
- 《君主论》第14章：将色诺芬的《居鲁士的教育》（Cyropaideia）列为君主应阅读的著作
- 《君主论》第6章：居鲁士被视为建国者的典范
- 《论李维》多处引用色诺芬的著作

**改造要点：**
- 色诺芬的居鲁士是理想君主的典范，马基雅维利将其改造为"必要性"驱动的政治家形象
- 马基雅维利对色诺芬的引用集中在军事和领导力方面

### 5.3 西塞罗（Marcus Tullius Cicero）

**引用频率：** 中等

**引用方式：**
- 《论李维》中多次引用西塞罗的《论共和国》（De Re Publica）和《论义务》（De Officiis）
- 《战争的艺术》的对话体形式模仿西塞罗

**改造要点：**
- 西塞罗强调政治中的道德约束，马基雅维利明确反对这一立场（《君主论》第15章）
- 但马基雅维利继承了西塞罗关于混合政体的思想
- 西塞罗的"国家理性"（ratio rei publicae）概念被马基雅维利发展为更激进的形式

### 5.4 普鲁塔克（Plutarch）

**引用频率：** 中等

**引用方式：**
- 《君主论》和《论李维》中引用希腊罗马名人传记
- 特别是阿基琉斯、忒修斯等人的事例

### 5.5 亚里士多德（Aristotle）

**引用频率：** 较低（直接引用不多，但影响深远）

**改造要点：**
- 马基雅维利不采用亚里士多德的政体分类法（君主/贵族/民主），而是采用更动态的分类
- 不接受亚里士多德的"人是政治动物"的自然主义假设——马基雅维利认为政治是人为创造的秩序
- 亚里士多德认为最好的政体是由中产阶级主导的混合政体，马基雅维利则更强调制度设计（ordini）的作用

### 5.6 塔西佗（Tacitus）

**引用频率：** 间接影响显著

**改造要点：**
- 塔西佗对帝制的冷静分析被马基雅维利借鉴
- 但马基雅维利更关注共和制度的建立，而非帝制的维持
- "塔西佗主义"（Tacitism）在16世纪的流行部分归功于马基雅维利的影响

### 5.7 圣经与宗教人物

**引用方式：**
- 摩西被视为最伟大的建国者（《君主论》第6章）
- 大卫王被视为有virtù的君主
- 教皇制度被系统性地批判

**改造要点：**
- 将宗教人物"去神圣化"，还原为政治人物
- 这是马基雅维利最具颠覆性的做法之一

## 八、学术争议与关键问题索引

### 8.1 马基雅维利是共和主义者还是君主制的支持者？

| 立场 | 代表学者 | 依据 |
|------|----------|------|
| 共和主义 | Pocock, Skinner, Viroli | 《论李维》是其真正信念的表达 |
| 君主制/权力政治 | Cassirer, Meinecke | 《君主论》是其核心著作 |
| 价值多元主义 | Isaiah Berlin | 两者都是真实的，体现了多元价值 |

### 8.2 《君主论》是讽刺还是字面建议？

| 立场 | 代表学者 | 依据 |
|------|----------|------|
| 字面建议 | Mansfield, Viroli | 书信中显示马基雅维利渴望被召回 |
| 讽刺/劝谏 | Mattingly, Pocock | 书中的荒谬建议（如应学习博尔贾，但博尔贾最终失败）暗示讽刺 |

### 8.3 马基雅维利的原创性

- 伯林（Berlin）：马基雅维利开创了"事实-价值分离"，是现代性的开端
- 斯特劳斯（Strauss）：马基雅维利是有意识地颠覆古典政治哲学
- 斯金纳（Skinner）：马基雅维利属于文艺复兴共和主义传统，并非完全原创
